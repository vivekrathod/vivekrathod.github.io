So, I have an iMac at home and a MacBook Air that I recently bought. I wanted to to be able to use iPhoto from both machines and have access to the same set of photos. This way I can import my photos into iMac and edit them later on from my Air, or vice verse.
I figured I could achieve this if I keep the iPhoto libraries in sync between the two machines. I started doing this with rsync, and it did work well. Something like this did the trick

rsync -avz  ~\Pictures/iPhoto\ Library/ user@imac.local:"~\Pictures/iPhoto\ Library/"
